# amaranthHDL-RXPowerMeter
RX Power Meter for the ADALM PLUTO SDR implemented in Python via AmaranthHDL
Designed as part of the University of Notre Dame's Wireless Institute's RadioHound Lab STRAND project

Validated via VCD analysis in Surfer in VSCode and Icarus Verilog synthesization check of the generated .v file

P = ($I^2$ + $Q^2$) / N (averaged over N samples)
